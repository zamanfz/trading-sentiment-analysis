# 📊 Trading Behavior Analysis using Fear & Greed Index

## 📌 Overview

This project analyzes how market sentiment (Fear vs Greed) affects trading behavior.

## 🎯 Objectives

* Analyze relationship between sentiment and profit (PnL)
* Study win rate variations
* Understand trading activity patterns

## 📂 Dataset

* Fear & Greed Index dataset
* Historical trading data

## ⚙️ Process

1. Data Cleaning (timestamp handling, merging)
2. Feature Engineering (win_rate, trade_count, daily_pnl)
3. Analysis & Visualization
4. Simple ML Model (Logistic Regression)

## 📊 Key Insights

* Greed leads to higher trading activity
* Fear results in cautious trading behavior
* Profitability varies with sentiment

## 🤖 Machine Learning

A simple Logistic Regression model is used to predict profitability.

## 🚀 Tech Stack

* Python
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

## 📎 How to Run

1. Open notebook in Google Colab or Jupyter
2. Upload datasets
3. Run all cells
